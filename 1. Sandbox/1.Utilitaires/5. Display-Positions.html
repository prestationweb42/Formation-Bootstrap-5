Display inline/block

Les éléments sont inline OU block par DEFAUT selon leur catégorie

div = d-block
div = d-block

On peut les display inline et prend ainsi la largeur du contenu

div = .d-inline - div = .d-inline
div = .d-inline

On peut gérer le display de manière responsive mais attention à l'affichage selon la longueur du contenu !

Exemple : d-inline d-md-block d-lg-inline

Les Positions

Les positions : position-static / relative / absolute / fixed / sticky

Les placements : top / left / right / bottom avec top-0 / 50 / 100

Le centrage avec : translate-middle ou translate-middle-x/y

Exemple : idem qu'en CSS, le parent en position-relative

Et les enfants position-absolute / position-fixed / position-sticky en %

Danger : position-absolute top-0 start-0

Success : position-absolute top-0 end-0

Primary : position-absolute bottom-0 start-0

Warning : position-absolute bottom-0 end-0

Dark 1 : position-absolute top-0 start-50 translate-middle-x

Dark 2 : position-absolute top-50 start-50 translate-middle
