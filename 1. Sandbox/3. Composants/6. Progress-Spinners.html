Créer un spinner

Sur le parent : spinner-border

Pour styliser : text-primary|secondary|danger|info|...m

Pour styliser : spinner-grow

Exemple avec : spinner-border text-primary

Exemple avec : spinner-grow

Créer des boutons spinner

Un bouton normal avec un spinner à l'intérieur

Sur le span pour en faire un petit : spinner-border-sm

Ici la classe : spinner-border est directement ajoutée sur le span vu qu'on l'utilise dans un bouton

Créer des barres de progressions

Sur le parent : progress

Sur l'enfant : progress-bar

Les attributs : aria/role servent à l'accessiblité

La width en style définit la largeur

Sur l'enfant, pour lui appliquer un style zébré : progress-bar-striped

Pour la rendre animée : progress-bar-animated

Exemple avec les classes : progress-bar progress-bar-striped progress-bar-animated bg-danger w-75
