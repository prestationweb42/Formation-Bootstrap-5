Les Carte

Une carte est un composant d'interface servant à montrer du contenu, qu'on réplique parfois (affichage d'articles, de profils multiples, etc ...)

Avec Bootstrap, elles sont composées de multiples classes, certaines n'apportant que peu de style.

Sur le parent : card

Styliser un header : card-header

Sur une image dans une carte : card-img-top

Sur le container du contenu : card-body

Sur les éléments du contenu : card-title|subtitle|text

Sur les liens : card-link

Notez la largeur en pixel, qui garde la même UI en permanence, mais qui empêche toute fluidité.
