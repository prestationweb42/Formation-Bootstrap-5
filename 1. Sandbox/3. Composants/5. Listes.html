Créer une liste

Sur le parent (ul) : list-group

Sur les enfants (li) : list-group-item

Si on veut surligner un élément enfant (li) : active

Pour le "désactiver" : disabled

Sur le parent (ul) pour numéroter les li : list-group-numbered

Pour les "styliser" : list-group-item-success|danger|primary|...
