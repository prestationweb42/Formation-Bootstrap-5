Utiliser Flexbox sans le système de Grille.

[d-flex]/ [d-inline-flex]

On peut aussi changer le display de manière réactive

Exemple : d-sm-flex, d-lg-flex, etc ...

Il existe ce genre de classes réactives pour toutes les propriétés liées à Flexbox.
